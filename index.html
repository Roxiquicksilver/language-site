Introduction to my website

If you would like to find out which are the best and maybe the worst learning resources for learning Japanese and Mandarin then you have come to the right place.

Also if you are a Japanese or Chinese speaker looking to find English learning resources, then look no further.

This site is a work in progress, so quite often the information on a certain resource will be incomplete. I aim to add to the site each week and hope that in the future this site can be used as a useful tool for language learners. Please post a comment if you see any spelling mistakes or would like to make a suggestion.
